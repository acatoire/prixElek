Description

Prise de courant quadruple 2P+T Legrand Céliane

Cette prise de courant Céliane, 2 pôles avec terre, de la marque Legrand, convient pour brancher 4 appareils électrodomestiques à la fois. Elle peut être utilisée dans le salon pour brancher, un lecteur DVD, radio, télévision ou pour recharger la batterie d’un téléphone portable. Elle peut également être utilisée dans la cuisine pour alimenter un robot mixeur, micro-onde, lave-vaisselle…en même temps. Cette prise de courant quadruple peut être associée avec un enjoliveur et avec une plaque de gamme Céliane.

Les + du produit :

  • Permet de disposer de 4 prises 2P+T dans le volume d'une boîte d'encastrement 3 postes
  • Permet en fonction de l'enjoliveur associé de réaliser des prises de courant Surface ou standard
  • Equipée de bornes automatiques avec repérage couleur

Caractéristiques du produit :

  • Programme Céliane™
  • Prises de courant, sorties de câble et obturateurs
  • Prises de courant précâblées équipées de bornes automatiques avec repérage couleur

Prise double

  • 4 x 2P+T précâblée
  • Permet en fonction de l'enjoliveur associé de réaliser des prises de courant Surface ou stanfard
  • Se monte sur support réf. 0 802 53 avec plaque spéciale 6/8 modules
  • Pour les choix de la plaque merci de voir sur la documentation Legrand 
  • Dimensions : 45 x 180 x 42 x 8.5 mm

Ce produit est compatible et peut être associé avec les produits de la nouvelle gamme Céliane

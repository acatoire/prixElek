##  Module 2 — Édition de devis *(V2, à affiner)*

### 1 Description

Permet de composer un devis client à partir du catalogue matériel, avec application d'un coefficient de marge.

### 2 Fonctionnalités envisagées

- Créer un nouveau devis (numéro auto, date, infos client)
- Ajouter des lignes matériel depuis le catalogue ou en saisie libre
- Appliquer un coefficient de marge par ligne ou global
- Ajouter des lignes de main-d'œuvre (taux horaire × temps)
- Générer un PDF ou document imprimable
- Archiver les devis

### 3 Lien avec le module Comparateur

- Lors de l'ajout d'un article, afficher le meilleur prix fournisseur comme base de calcul
- Possibilité de lier le devis à un panier fournisseur
